Качаем сайт termux wget

Последнее время сайты закрываются и имеет смысл сохранить их себе на локальный диск, дабы не потерять информацию.

Довольно простой способ это сделать прямо на телефоне это использовать wget в терминале termux

pkg install wget

После установке можно переходить к скачиванию сайта

wget -m -l 10 -e robots=off -p -k -E --reject-regex "wp" --no-check-certificate -U="Mozilla/5.0 (Windows NT 10.0; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/97.0.4692.99 Safari/537.36" https://site.com

OПЦИИ

-m - Для неприхотливого хранения зеркальных копий сайтов вы можете использовать данный параметр.

-l - определяет максимальную глубину вложенности страниц, которые wget должен скачать (по умолчанию 5)

-e robots=off - Игнорирует robot.txt.

-p - Загружать все файлы, которые нужны для отображения страниц HTML. Например: рисунки, звук, каскадные стили.

-k - После завершения загрузки конвертировать ссылки в документе для просмотра в автономном режиме.

-E - добавлять к загруженным файлам расширение .html

-U - юзерагент

Вот скрипт который использую для скачивания с помощью wget в termux

cd storage/downloads/site1/;
echo "url site:";read link;
wget -m -l 10 -e robots=off -p -k -E --reject-regex "wp" --no-check-certificate -U="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/98.0.3440.106 Safari/537.36" "$link";

Более подробную информацию можете узнать по команде:

wget - h
