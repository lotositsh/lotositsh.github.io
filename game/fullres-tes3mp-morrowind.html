Запуск Fullrest Repack в Tes3mp: Играем в Morrowind онлайн



В данной статье я расскажу о том, как запустить сборку Fullrest Repack в Tes3mp для игры в Morrowind онлайн.

Для начала, необходимо скачать и установить сборку The Elder Scrolls III: Morrowind Fullrest Repack версии 4.0.00, доступную по ссылке

The Elder Scrolls III: Morrowind [Fullrest Repack]

К сожалению, сборка новее версии не запустилась по техническим причинам.

После установки сборки необходимо скачать Tes3mp 0.8.1, который доступен по ссылке

TES3MP

Распакуйте файлы в удобную для вас директорию и запустите openmw-launcher.exe. Укажите путь к игре: MFR\game\Data Files\Morrowind.esm и нажмите "добавить". Моды и игра должны быть загружены автоматически.

Теперь проставляем моды в порядке который указан на изображении, отключаем TR_patch.esp.



Переходим в папку TES3MP\server\data\ и редактируем файл requiredDataFiles.json В нем нужно указать все моды которые должны быть на сервере и клиенте. У меня получилось так(сохраните последовательность):

Следующим шагом будет настроить сам сервер. Я покажу как настроить на локальном примере.

Для сервера редактируем в папке TES3MP файл tes3mp-server-default.cfg

localAddress = 192.168.0.106
port = 25565
maximumPlayers = 4
hostname = aloha
# 0 - Verbose (spam), 1 - Info, 2 - Warnings, 3 - Errors, 4 - Only fatal errors
logLevel = 1
password = 21

[Plugins]
home = ./server
plugins = serverCore.lua

[MasterServer]
enabled = false
address = master.tes3mp.com
port = 25561
rate = 10000

Для клиента tes3mp-client-default.cfg

[General]
destinationAddress = 192.168.0.106
port = 25565
password = 21
# 0 - Verbose (spam), 1 - Info, 2 - Warnings, 3 - Errors, 4 - Only fatal errors
logLevel = 0

[Master]
address = master.tes3mp.com
port = 25561

[Chat]
# Use https://wiki.libsdl.org/SDL_Keycode to find the correct key codes when rebinding
#
# For chatting
keySay = Y
# For enabling or disabling the chat window
keyChatMode = F2
x = 0
y = 0
w = 390
h = 250
# How long the message will be displayed in hidden mode
delay = 5.0

Запускаем сервер tes3mp-server.exe, запускаем игру tes3mp.exe. Вводим логин и пароль. Готово.





Стоит помнить что данный пример для локальной сети или через radmin vpn, для открытого сервера нужно открыть порт 25561, заменить 

[MasterServer]
enabled = true

для отображения вашего сервера в окне браузера всех серверов.

Настройка сервера

Основные настройки сервера найдете в файле TES3MP\server\scripts\config.lua Там мы сможем настроить общий журнал или нет и много другой полезной информации.

Теперь по квестам. Если у вас разные журналы, диалоги и квесты. Второй игрок заходит, а квестового предмета нет, то нужно удалить содержимое и перезапустить сервер. TES3MP\server\data\cell\ . Это сохранит общий прогресс всех игроков

Если хотите сбросить сервер (журнал тоже) то удалите все в папке TES3MP\server\data\world\ или запустите tes3mp-server-autorestart.bat

P.S. Сжал и скинул игру в архиве на гугл диск. Расположение файлов для запуска:

M[FR]\M[FR] Launcher.exe - основной лаунчер для игры mcp/mge/openmw для игры оффлайн

M[FR]\game\TES3MP\openmw-launcher.exe - для игры онлайн/сети

M[FR]\game\TES3MP\ - в этой папке также файлы конфигурации сервера и клиента

M[FR]\game\TES3MP\server\data\requiredDataFiles.json - файл настройки сервера

M[FR]\game\TES3MP\server\data\world\ - файлы сохраненного мира

M[FR]\game\TES3MP\server\data\cell\ - квесты, события и предметы

Место для распаковки 14 Гб (Сжат до 6 Гб), для распаковки использовать 7z или другой архиватор с поддержкой .7z
