Baldur's Gate 3: как перенести папку с модами на другой диск

Сейв файлы и моды находятся по пути %localappdata%\Larian Studios

(просто скопируй и вставь в папку) будет такое расположение:

C:\Users\YOU_USER_NAME\AppData\Local\Larian Studios

YOU_USER_NAME - имя пользователя Windows

Создаем папку куда будем переносить файлы

F:\bg3

Копируем всё содержимое из %localappdata%\Larian Studios\Baldur's Gate 3 в новую папку F:\bg3.

Папку Baldur's Gate 3 из %localappdata%\Larian Studios\ можно удалить.

Теперь открывает командную строку (win+R пишем cmd) или через поиск запускаем командную строку через администратора.

Переходим в нужную папку:

cd /d %localappdata%\Larian Studios

Создаем символьную ссылку для игры

mklink /j "Baldur's Gate 3" "F:\bg3"

После этого в исходной паке %localappdata%\Larian Studios вы увидете папку Baldur's Gate 3, которая является ссылкой на файлы F:\bg3.
