Termux создание новой вкладки

Для создания новой сессии в Termux (открыть новую вкладку), нужно перейти в домашний каталог

cd~
создать новую папку 
mkdir .termux
cd .termux
nano termux.properties

Если нет редактора nano устанавливаем

apt install nano
прописать следующую команду 
# Open a new terminal with ctrl + t (volume down + t)
shortcut.create-session = ctrl + t
# Go one session down with (for example) ctrl + 2
shortcut.next-session = ctrl + 2
# Go one session up with (for example) ctrl + 1
shortcut.previous-session = ctrl + 1
# Rename a session with (for example) ctrl + n
shortcut.rename-session = ctrl + n

выходим

exit
Теперь при нажатии сочетаний клавиш ctrl+t будет созданная новая сессия в termux, перемещаться между сессиями можно при помощи ctrl + 1 и ctr + 2 тд
