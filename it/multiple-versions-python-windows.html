Несколько версий Python для Windows: установка и использование

1. Скачать Python

Выберите нужную версию на официальном сайте.

2. Установка

Запустите установщик, снимите галочку Add Python to PATH (если нужно избежать конфликтов).

3. Проверка версий

py -0  # список установленных версий и текущей по умолчанию
py -3.12

4. Создание виртуального окружения

py -3.12 -m venv venv  
venv\Scripts\activate

5. Проверка версии Python

python -V  # текущая версия в окружении
deactivate
