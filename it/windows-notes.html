Подключение к сети wifi через cmd

просмотр сетей

netsh wlan show networks


Создаем папку и файл будущего профиля сети

C:\WiFiProfiles\NameWIFI.xml

Содержание файла:

	<?xml version="1.0"?>
	<WLANProfile xmlns="http://www.microsoft.com/networking/WLAN/profile/v1">
		<name>NameWIFI</name>
		<SSIDConfig>
			<SSID>
				<name>NameWIFI</name>
			</SSID>
		</SSIDConfig>
		<connectionType>ESS</connectionType>
		<connectionMode>auto</connectionMode>
		<MSM>
			<security>
				<authEncryption>
					<authentication>WPA2PSK</authentication>
					<encryption>AES</encryption>
					<useOneX>false</useOneX>
				</authEncryption>
				<sharedKey>
					<keyType>passPhrase</keyType>
					<protected>false</protected>
					<keyMaterial>PASSWORD</keyMaterial>
				</sharedKey>
			</security>
		</MSM>
	</WLANProfile>
NameWIFI.xml

Важно учитывать типы безопасности:

	Типы безопасности: WPA2 Enterprise или WPA2 Personal
	Типы шифрования: AES или TKIP
	Типы EAP: смарт-карта или другой сертификат или PEAP

Добавляем профиль:

netsh wlan add profile filename="C:\WiFiProfiles\NameWIFI.xml"

Проверяем его наличие:

netsh wlan show profiles

Подключаемся:

netsh wlan connect name="NameWIFI"

Отключаемся:

netsh wlan disconnect

Вот небольшой баник

	@echo off
chcp 65001 > nul

:menu
echo.
echo ======================
echo 1. Сканировать сеть
echo 2. Показать доступные Wi-Fi профили
echo 3. Подключиться к профилю NameWIFI
echo 4. Подключиться к профилю 2
echo 5. Отключиться от Wi-Fi
echo 6. Выйти
echo ======================
echo.
set /p choice="Выберите действие: "

if "%choice%" == "1" goto scan
if "%choice%" == "2" goto show_profiles
if "%choice%" == "3" goto connect_NameWIFI
if "%choice%" == "4" goto connect_2
if "%choice%" == "5" goto disconnect
if "%choice%" == "6" exit

echo Неверный выбор. Попробуйте снова.
goto menu

:scan
echo Сканирование сети...
netsh wlan show networks
pause
goto menu

:show_profiles
echo Доступные Wi-Fi профили:
netsh wlan show profiles
pause
goto menu

:connect_NameWIFI
echo Подключение к профилю NameWIFI...
netsh wlan connect name="NameWIFI"
pause
goto menu

:connect_2
echo Подключение к профилю 2...
netsh wlan connect name="2"
pause
goto menu

:disconnect
echo Отключение от Wi-Fi...
netsh wlan disconnect
pause
goto menu
