Как выйти из аккаунта Git на Windows 10

Добрый день! В этой статье я расскажу вам, как выйти из аккаунта Git на Windows 10. Это может быть полезно в случае, если вы хотите переключиться на другую учетную запись или просто завершить сеанс.

Чтобы выйти из аккаунта Git на Windows 10, выполните следующие шаги:

  1. Откройте "Панель управления". Вы можете сделать это, щелкнув правой кнопкой мыши на кнопке "Пуск" в нижнем левом углу экрана и выбрав в появившемся контекстном меню пункт "Панель управления".


  2. Внутри "Панели управления" найдите и откройте раздел "Учетные записи пользователей". Вы можете воспользоваться поиском в верхнем правом углу окна, чтобы найти его быстрее.


  3. После открытия "Учетных записей пользователей" вам может потребоваться ввести учетные данные администратора Windows 10, чтобы получить права доступа.


  4. Внутри "Управление учетными данными Windows" найдите и выберите учетную запись, связанную с аккаунтом Git, из которого вы хотите выйти.


  5. Щелкните на выбранной учетной записи и выберите опцию "Изменить" или "Удалить". В зависимости от вашего выбора, вы можете изменить настройки учетной записи или удалить ее полностью.

ps. Более простой способ попасть в управление данными это просто вбить в поиск "Управление учетными данными Windows"



После выполнения этих шагов, вы успешно выйдете из аккаунта Git на Windows 10. Теперь вы можете войти в другой аккаунт или завершить сеанс.

Надеюсь, эта инструкция была полезной для вас.
