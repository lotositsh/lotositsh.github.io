SSH автоматическое переподключение

Для автоматического восстановления соединения с помощью SSH при обрывах я использую Bitvise SSH Client. В отличие от PuTTY, он умеет поддерживать соединение. Вот пример настройки для PuTTY, аналогичный настройке в Bitvise SSH Client, для лучшего понимания:


putty.exe -ssh -D 8889 u0_a288@192.168.0.102 -P 8022 -pw 4YWkL27iV0Bx





Скачать можно на офф. сайте или как альтернатива в телеграмме
