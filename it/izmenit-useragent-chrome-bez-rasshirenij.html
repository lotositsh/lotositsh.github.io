Как изменить user agent chrome без расширений

Чтобы изменить user agent в браузере Chrome без дополнительных расширений достаточно создать ярлык браузера и в свойствах указать параметр

 --user-agent=""

В кавычках указываем нужный нам useragent. Пример:

--user-agent="Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/79.0.3945.117 Safari/537.36 Edg/79.0.309.65"

В этом примере мы задали операционную систему MacOS с браузером Chrome версии 79

Таким образом можно создать несколько ярлыков с разными useragent под свои нужды. В системе Linux всё делается точно также, открываем терминал, пишем

chrome --user-agent=""

Проверить полученный результат можно на сайте whoer.net
