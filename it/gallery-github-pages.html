Создание красивого Git репозитория для сайта галереи

Создаем Git с красивым именем для будущего сайта галереи.

Переходим в репозиторий

[ссылка].

Нажимаем "Use this template" -> "Create new repository" -> "name.github.io" -> "Create"



Ждем и проверяем, появился ли сайт по ссылке "name.github.io".

Развертку можно проверить в "Actions".

Далее, в "Settings" -> "Pages" -> "Build and deployment" -> "Github Actions"



В файле "config.json" меняем название "footer" на "Свое название галереи". >Создаем папку "name.github.io/gallery/folder_name/.gitkeep"

.

Добавляем в папку изображения с соответствующими названиями, и галерея готова.

После добавления, галерея будет пересобрана в Github Actions каждый раз.



Имейте в виду, что есть лимиты: один файл не может быть больше 75 МБ, и гит рекомендует не превышать 1 ГБ для одного репозитория.

Для загрузки большого числа файлов лучше использовать Git Bash.

Также, рекомендуется загружать все сразу, поскольку добавление readme-файла вызывает заново создание галерей, что может быть нагрузкой .
